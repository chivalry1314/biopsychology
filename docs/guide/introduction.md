# 项目简介

**生理心理学全息交互诊断分析终端**（Bio-Psychology Quantum Visualizer）是一个面向科普与教学的交互式可视化应用。它通过 3D 图形、实时动画和数据可视化，将抽象的生理心理学概念转化为可感知、可操作的界面。

## 设计目标

- **科普教育**：帮助用户直观理解大脑结构、神经递质与情绪之间的关系。
- **教学演示**：为心理学、神经科学、生理学课程提供动态演示素材。
- **交互探索**：通过滑块、按钮和场景预设，让用户亲手调节参数并观察结果。

## 核心特色

1. **量子仪表盘风格**：深色主题、霓虹配色、终端式 UI，营造未来实验室氛围。
2. **多维度可视化**：3D 点云大脑、Canvas 动画、Chart.js 图表、SVG 示意图相结合。
3. **临床案例驱动**：内置多种典型神经精神疾病案例，一键加载对应参数。
4. **纯前端实现**：无需后端服务，构建产物为静态文件，部署简单。
