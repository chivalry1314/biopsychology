# 🎁 奖励通路与多巴胺奖赏

中脑边缘多巴胺通路是大脑奖赏系统的核心。本模块可视化腹侧被盖区（VTA）到伏隔核与前额叶皮层的奖赏通路，帮助理解自然奖赏、预期、成瘾与戒断的神经机制。

## 主要功能

### 奖赏通路动画

使用 SVG 动画展示三条关键节点：

- **VTA（腹侧被盖区）**：多巴胺神经元胞体所在区域。
- **伏隔核（Nucleus Accumbens）**：奖赏感受与动机驱动的关键区域。
- **前额叶皮层（Prefrontal Cortex）**：负责决策、冲动控制与预期评估。

动画中多巴胺信号沿通路传递，亮度与频率随状态变化。

### 奖赏预测误差（RPE）仪表

- 展示实际奖赏与预期奖赏之间的差异。
- 意外大奖导致正 RPE，增强学习与动机。
- 预期落空导致负 RPE，产生失望或更新预期。

### 多巴胺 / 渴求 / 满足 曲线

实时绘制三条动态曲线：

- **多巴胺水平**：反映奖赏信号的强度。
- **渴求强度**：反映对奖赏的动机性追求。
- **满足程度**：反映奖赏获得后的满足感。

### 耐受指数与戒断反应

模拟成瘾行为中的两个关键现象：

- **耐受**：反复暴露后，相同刺激产生的多巴胺反应逐渐减弱。
- **戒断**：停止刺激后，多巴胺水平骤降，出现负性情绪与强烈渴求。

### 场景预设

- **基线**：正常静息状态。
- **自然奖赏**：食物、社交等自然奖励。
- **预期之内**：奖赏与预期一致。
- **意外大奖**：超出预期的高额奖赏。
- **成瘾物质**：模拟物质滥用导致的强烈多巴胺释放。
- **戒断反应**：停止使用后的低落与渴求状态。

## 使用场景

- 理解多巴胺在动机与学习中的作用。
- 解释成瘾行为的神经生物学基础。
- 讨论奖赏预测误差与决策行为的关系。
