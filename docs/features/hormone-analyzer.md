# 🧪 递质情绪协同分析

递质情绪协同分析模块将多种神经递质的综合影响转化为可视化的多维心理状态空间，帮助用户理解递质水平与情绪、行为之间的复杂关系。

## 主要功能

### 六维心理状态雷达图

基于当前递质水平，计算并展示 6 个心理维度：

| 维度 | 含义 |
|------|------|
| 行为驱动 | 动机、目标导向行为的强度 |
| 焦虑应激 | 紧张、警觉、应激反应水平 |
| 感官欣快 | 愉悦感、奖赏感受的强度 |
| 抑郁倾向 | 情绪低落、快感缺失的倾向 |
| 自我稳定 | 情绪稳定与自我调节能力 |
| 突触可塑性 | 学习与记忆相关的神经可塑性 |

### 自主神经系统平衡指数

计算并展示交感神经系统与副交感神经系统的相对张力：

- **交感神经**：主导战斗或逃跑反应，提升心率、血压与警觉性。
- **副交感神经**：主导休息与消化，促进放松与恢复。

平衡指数帮助判断当前状态是偏亢奋、偏放松还是相对平衡。

### 临床心理解读

根据雷达图与平衡指数，生成简短的文字解读，描述当前心理状态的可能特征。

## 使用场景

- 分析不同递质组合对情绪的影响。
- 理解焦虑、抑郁、欣快等状态背后的神经生物学基础。
- 与临床案例模块联动，观察疾病状态下的心理特征。
