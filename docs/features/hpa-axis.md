# 🏔️ HPA 轴压力反应

下丘脑-垂体-肾上腺轴（HPA 轴）是人体应对压力的核心内分泌系统。本模块可视化压力刺激下 HPA 轴的级联反应以及皮质醇水平的动态变化。

## 主要功能

### HPA 轴级联可视化

展示从压力感知到激素释放的完整路径：

1. **下丘脑（Hypothalamus）**：感知压力，释放 CRH。
2. **垂体（Pituitary）**：接收 CRH，释放 ACTH。
3. **肾上腺（Adrenal Gland）**：接收 ACTH，释放皮质醇。

动画中可观察到信号沿该通路逐级传递的过程。

### 皮质醇动态曲线

使用 Chart.js 折线图展示皮质醇浓度随时间的变化：

- 压力源出现后皮质醇迅速上升。
- 持续压力导致皮质醇维持高位。
- 慢性压力模式下可能出现失调曲线。

### 压力源强度调节

通过滑块调节压力源的强度，观察 HPA 轴反应的剧烈程度：

- 轻度压力：皮质醇小幅上升，很快恢复。
- 中度压力：明显上升并持续一段时间。
- 强烈压力：大幅上升，恢复缓慢。

### 慢性压力模式

开启慢性压力后，系统模拟长期压力下的 HPA 轴功能变化：

- 皮质醇基线升高。
- 反馈调节能力下降。
- 身体进入持续高负荷状态。

### GAS 阶段指示

根据皮质醇水平和持续时间，显示一般适应综合征（GAS）阶段：

- 基线
- 警觉期
- 抵抗期
- 衰竭期

### 生理影响指标

实时显示压力反应对身体的多方面影响：

- 血糖水平
- 免疫抑制程度
- 抗炎水平
- 去甲肾上腺素联动

## 使用场景

- 学习压力反应的神经内分泌机制。
- 理解急性压力与慢性压力的区别。
- 讨论压力相关疾病（如慢性疲劳、焦虑障碍）的生理基础。
