# 📊 生理化学递质调控面板

生理化学递质调控面板是应用的核心控制台之一。用户可以像调节合成器一样，实时调整 4 种关键神经递质的水平，并即时观察主导心理状态的变化。

## 主要功能

### 神经递质滑块

| 递质 | 主要作用 |
|------|---------|
| 多巴胺 | 奖赏、动机、运动控制 |
| 5-羟色胺 | 情绪稳定、睡眠、食欲 |
| 去甲肾上腺素 | 警觉、注意力、应激反应 |
| 内啡肽 | 镇痛、愉悦、放松 |

每个递质都有独立的滑块，可自由调节水平高低。

### 主导心理状态

根据当前各递质的相对水平，界面会实时显示一种主导心理状态，例如：

- 高度警觉
- 平静放松
- 愉悦兴奋
- 焦虑紧张
- 抑郁低落

### 情绪预设模式

提供三种一键场景预设：

- **激悦模式**：提升多巴胺与内啡肽，模拟积极兴奋状态。
- **镇静沉思**：提升 5-羟色胺，降低去甲肾上腺素，模拟冥想放松状态。
- **战斗或逃跑**：大幅提升去甲肾上腺素，模拟急性应激状态。

## 使用场景

- 探索不同递质水平对主观体验的影响。
- 为临床案例设置初始递质参数。
- 配合其他模块（脑电、HPA 轴、自主神经）观察连锁反应。
