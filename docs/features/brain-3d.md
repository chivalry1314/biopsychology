# 🧠 3D 大脑与功能定位

3D 大脑模块是整个应用的视觉核心。它使用 Three.js 将真实的大脑顶点数据渲染为点云，并按空间位置划分为 6 个主要脑区，帮助用户建立大脑结构与功能的对应关系。

## 主要功能

### 点云大脑渲染

- 加载 `public/models/brain_vertex_low.obj` 中的真实大脑顶点数据。
- 将顶点按空间坐标分类为 6 大功能区：额叶、颞叶、顶叶、枕叶、小脑、脑干。
- 使用粒子系统渲染，每个顶点对应一个发光点，形成全息大脑效果。

### 交互操作

| 操作 | 效果 |
|------|------|
| 鼠标拖拽 | 旋转大脑模型 |
| 滚轮 | 缩放视图 |
| 点击脑区 | 高亮选中区域并在右侧面板显示详情 |
| 触摸手势 | 在移动设备上同样支持旋转与缩放 |

### 视觉效果

- **区域高亮**：选中某个脑区时，该区域点云增强发光，其他区域变暗。
- **神经通路连线**：在相关脑区之间绘制连线，模拟神经纤维束。
- **信号粒子流动**：发光粒子沿通路在大脑不同区域间流动，模拟神经冲动传导。
- **自动旋转**：可开启/关闭大脑的缓慢自动旋转展示。

### 参数调节

- 点大小
- 亮度
- 背光强度
- 轮廓光强度
- 自动旋转开关

## 脑区详情面板

点击任意脑区后，右侧面板 `RegionDetail.vue` 会展示该区域的：

- **生理学解读**：位置、结构、主要功能。
- **心理学效应**：与情绪、认知、行为的关系。
- **临床病理**：相关疾病的简要说明。
- **关键神经递质**：该区域主要涉及的神经递质类型。

## 使用场景

- 课堂教学中展示大脑分区。
- 自学神经科学时快速查阅各脑区功能。
- 临床案例分析前定位相关脑区。
