# 🌊 脑电节律模拟器

脑电图（EEG）记录大脑神经元的电活动。本模块通过 Canvas 实时生成五种典型脑电节律的波形，并展示各频段的能量分布。

## 主要功能

### 五种脑电频段

| 频段 | 频率范围 | 典型状态 |
|------|---------|---------|
| δ（Delta） | 0.5-4 Hz | 深睡、无意识 |
| θ（Theta） | 4-8 Hz | 冥想、浅睡、创造力 |
| α（Alpha） | 8-13 Hz | 放松、闭眼休息 |
| β（Beta） | 13-30 Hz | 专注、警觉、思考 |
| γ（Gamma） | 30-100 Hz | 兴奋、认知整合 |

### 实时波形动画

- Canvas 动态绘制当前主导频段的脑电波形。
- 波形频率、振幅随递质水平变化而调整。

### 频段能量分布

以条形图展示 5 种频段的相对能量占比，帮助判断当前大脑的活跃模式。

### 自动计算主导频率

模块会根据神经递质调控面板中的递质水平，自动计算当前最可能的主导 EEG 频率。例如：

- 高多巴胺、高去甲肾上腺素 → β / γ 增强
- 高 5-羟色胺、高内啡肽 → α / θ 增强

## 使用场景

- 理解不同意识状态对应的脑电特征。
- 观察递质变化如何影响脑电节律。
- 结合睡眠、冥想、专注等主题进行教学演示。
