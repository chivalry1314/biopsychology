# 💓 自主神经系统平衡仪

自主神经系统（Autonomic Nervous System, ANS）调节心率、呼吸、消化、出汗等不自主生理功能。本模块通过可视化仪表与人体示意图，帮助用户理解交感神经与副交感神经之间的动态平衡。

## 主要功能

### 交感 / 副交感神经滑块

- **交感神经张力**：代表身体的应激动员能力。
- **副交感神经张力**：代表身体的休息恢复能力。

通过滑块调节两者比例，观察身体状态的变化。

### Canvas 平衡仪表

以仪表盘形式直观展示当前自主神经系统的平衡状态：

- 偏向交感神经：进入警觉、应激状态。
- 偏向副交感神经：进入放松、恢复状态。
- 居中表示相对平衡。

### 人体 SVG 示意图

高亮显示受自主神经影响的主要器官：

- 大脑
- 瞳孔
- 心脏
- 肺
- 汗腺
- 消化系统

### 实时生理指标

根据交感神经与副交感神经的张力，实时计算并显示：

- 心率
- 心率变异性（HRV）
- 呼吸频率
- 瞳孔直径
- 皮肤电反应
- 消化活性

### 场景预设

- **深度放松**：副交感神经主导，心率下降，呼吸缓慢。
- **考试压力**：交感神经轻度升高，警觉增强。
- **剧烈运动**：交感神经高度激活，心率与呼吸显著上升。
- **惊恐发作**：交感神经极度亢进，出现强烈应激反应。

## 使用场景

- 理解自主神经系统如何调节日常生理状态。
- 学习压力、放松、运动等场景下的身体反应机制。
- 配合临床案例分析自主神经功能紊乱。
