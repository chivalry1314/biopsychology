# ⚡ 突触递质传递模拟

突触是神经元之间信息传递的基本结构。本模块通过 Canvas 2D 动画，将微观尺度的突触传递过程可视化，让用户可以直观地观察动作电位、囊泡释放与受体结合。

## 主要功能

### 突触结构展示

动画中包含三个关键区域：

- **突触前膜**：含有神经递质囊泡的神经元末端。
- **突触间隙**：前后膜之间的狭小空隙。
- **突触后膜**：带有受体的下一个神经元表面。

### 递质释放

用户可以主动触发以下递质的释放：

- 多巴胺
- 血清素（5-HT）
- 去甲肾上腺素

释放后，囊泡向突触间隙移动并扩散，最终与突触后膜上的受体结合。

### 动作电位传播

- 动画展示电信号沿轴突传播到突触前膜的过程。
- 可实时观察动作电位的触发与传导。

### 实时指标

界面实时显示：

- 动作电位强度
- 囊泡释放速率
- 受体结合率

### 音频反馈

通过 Web Audio API 提供音效反馈，增强沉浸感。部分浏览器需要用户先与应用交互才能启用声音。

## 使用场景

- 学习神经信号传递的基本原理。
- 理解不同神经递质在突触层面的作用差异。
- 配合临床案例，观察递质异常如何影响神经传递。
