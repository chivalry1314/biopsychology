# 🏥 临床案例模拟

临床案例模块将抽象的神经生化参数与具体疾病联系起来。用户可以通过一键加载，观察不同疾病状态下的递质水平、脑区活动与心理状态变化。

## 主要功能

### 内置临床案例

| 案例 | 核心特征 |
|------|---------|
| **重度抑郁症** | 多巴胺、5-羟色胺水平偏低，抑郁倾向维度升高 |
| **ADHD 多动症** | 多巴胺与去甲肾上腺素调节异常，注意力维度波动 |
| **精神分裂症** | 多巴胺功能亢进，认知与自我稳定维度异常 |
| **成瘾行为** | 奖赏通路高度敏感，渴求强而满足弱 |

### 一键加载

点击案例卡片后，系统会自动：

1. 设置神经递质调控面板到对应水平。
2. 高亮相关脑区。
3. 更新雷达图、自主神经平衡、脑电节律等模块。
4. 跳转到对应脑区或功能模块的详细视图。

### 跨模块联动

临床案例不是孤立模块，而是贯穿整个应用的起点：

- 在 **3D 大脑** 中定位相关脑区。
- 在 **递质调控面板** 中观察生化参数。
- 在 **情绪协同分析** 中查看心理状态雷达图。
- 在 **HPA 轴** 或 **自主神经** 模块中观察伴随的生理反应。

## 使用场景

- 课堂上演示神经精神疾病的生化基础。
- 自学时通过案例快速建立疾病与脑机制的关联。
- 作为讨论药物治疗靶点（如 SSRI、多巴胺调节剂）的切入点。
